# 管理學

### 企業是什麼?
#### 企業 是由個人或一群人組成

主要活動
* 取得並運用生產要素，進行產品或服務的交易與銷售，滿足消費者需求並藉此獲取利潤

#### 生產要素

1. 自然資源
2. 人力資源
    * 勞心或勞力的工作者
3. 資本設備
    * 人造的{生產工具}
4. 企業家精神
    * 創業精神

#### 滿足消費者需求的產品與服務

1. 有形的產品    
2. 無形的服務 -> 嘉惠客戶的活動 : 服務業

### 企業功能與管理功能



| 企業功能 | 管理功能 | 
| -------- | -------- |
| 生產功能     | 規劃     | 
| 行銷功能     | 組織     | 
| 人力資源功能     | 領導     | 
| 研究與發展功能     | 控制     | 
| 財務功能     |      | 
| 資訊功能     |      | 

規劃: 未來組織績效訂出目標

組織: 任務指派、任務部門化、資源分派
針對組織目標、偵測、比較、評估及改正的程序


領導： 運用影響力，激勵員工達成組織的目標

控制：對員工或事業單位的績效衡量，以確定組織的計畫或目標的實現並視情況決定是否採取調整行為

### 效果 vs 效率

在同樣努力下，第一次就做好
->高效率高效果

效率是指投入與產出的關係

兩者相互沒有關係

### 技能

概念技能：管理者須具備對問題的解決能力與思考能力，以正確分析情境並做出決策
